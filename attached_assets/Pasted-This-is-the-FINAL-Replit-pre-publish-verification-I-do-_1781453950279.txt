Verify these formulas exactly:

Required System Size = Annual Usage kWh ÷ Peak Sun Hours ÷ 365 ÷ 0.78

Annual Production = System Size kW × Peak Sun Hours × 365 × 0.78

Default panel wattage = 440W

Panel Count = Required System Size kW × 1000 ÷ 440

Panel count must round up.

Final System Size = Panel Count × 440 ÷ 1000

Estimated Annual Production = Final System Size × Peak Sun Hours × 365 × 0.78
