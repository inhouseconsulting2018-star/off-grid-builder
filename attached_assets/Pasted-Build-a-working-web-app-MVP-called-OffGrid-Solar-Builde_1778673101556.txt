3. Calculation Engine
Create working solar calculations using reasonable assumptions.

Calculate:
- average daily kWh usage = annual kWh / 365
- estimated peak sun hours based on state or ZIP placeholder
- solar array size in kW
- number of panels required using 400W default panels
- system losses from inverter loss, wire loss, shading loss, temperature loss, battery loss, dirt/soiling loss
- adjusted solar size after losses
- inverter size recommendation
- battery usable kWh requirement based on backup hours
- total battery bank size using 80% depth of discharge
- estimated equipment cost
- estimated DIY cost
- estimated installed cost
- grid-tied payback estimate
- yearly production estimate

Use these default assumptions:
- panel wattage: 400W
- base system loss: 14%
- inverter loss: 4%
- wire loss: 2%
- dirt/soiling loss: 3%
- temperature loss: 5%
- battery round trip loss: 10% when batteries are selected
- shade loss: none 0%, light 5%, medium 15%, heavy 30%
- battery depth of discharge: 80%
- economy cost: $1.25/watt DIY, $2.75/watt installed
- mid-range cost: $1.75/watt DIY, $3.25/watt installed
- premium cost: $2.25/watt DIY, $4.00/watt installed
- default utility rate: user input or $0.35/kWh
